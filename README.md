# My First Project

A modern HTML + JavaScript project scaffolded with an industry-standard structure and npm support using Vite.

## Prerequisites

- Node.js 14+
- npm 6+

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview the production build:

```bash
npm run preview
```

## Project Structure

```text
my-first-project/
	public/                # Static assets copied as-is
	src/
		styles/
			main.css           # App styles
		main.js              # JavaScript entry point
	index.html             # HTML entry file
	package.json           # npm scripts and dependencies
	.gitignore
	README.md
```
